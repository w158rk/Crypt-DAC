# 数据

```mermaid
classDiagram
    class User {
        -Set~UR~ roles
    }

    class UR {
        -User user
        -Role role
    }

    class Role {
        -Set~UR~ roles
        -Set~PA~ pas
    }

    class PA {
        -Role role
        -Perm perm
    }

    class Perm {
        -Set~PA~ pas
    }

    User "1" .. "n" UR
    Role "1" .. "n" UR
    Role "1" .. "n" PA
    Perm "1" .. "n" PA
```
