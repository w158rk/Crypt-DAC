## 关于项目


复现Crypt-DAC，动态访问控制的仿真


### 构建

gradle build


<!-- GETTING STARTED -->
## 入门


### 先决条件

- 安装Java和gradle

### 安装

1. 克隆仓库
   ```sh
   git clone https://github.com/w158rk/Crypt-DAC.git
   ```


<!-- USAGE EXAMPLES -->
## 使用

```sh
gradle run
```
